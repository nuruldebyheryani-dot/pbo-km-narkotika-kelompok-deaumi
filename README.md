# pbo-km-narkotika-kelompok-deaumi
Project tugas besar Pemrograman Berorientasi Object (PBO) berupa Knowledge Management System Putusan Pengadilan Narkotika menggunakan Java, MVC, dan GitHub collaboration.

## Fitur Utama

- Menampilkan daftar putusan pengadilan narkotika
- Menambahkan data putusan
- Mencari putusan berdasarkan nomor perkara
- Mencari putusan berdasarkan nama terdakwa
- Menghapus data putusan
- Menampilkan statistik putusan
- Menghitung total denda dari seluruh putusan

## Struktur Repository

src
├── app
├── controller
├── model
├── view
└── util

## Cara Menjalankan Program

1. Buka project menggunakan Intellij IDEA.
2. Jalankan file 'src/app/main.java'.
3. Gunakan menu yang tersedia pada aplikasi.
